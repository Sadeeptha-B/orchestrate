# ⏱ 2. The Transition Rules (this is what makes it work)

Think of this as a **state machine**, not vibes.

---

## ▶️ Step 1: Starting work

* Play **Start Work playlist**
* Do *anything related to the task* (even messy)

⏱ Duration:
👉 **5–10 minutes OR until you feel “locked in enough”**

**Transition when:**

* resistance drops slightly
* you’ve begun actual task interaction

---

## 🔁 Step 2: Switch to task-specific audio

Immediately switch based on task:

| Task                     | Switch to      |
| ------------------------ | -------------- |
| Coding / problem solving | Deep Focus     |
| Lectures / passive input | Lo-Fi          |
| Restless state           | Brain Food     |
| Low energy               | Peaceful Piano |

👉 Don’t stay in the start playlist too long — it’s just a ramp, not a workspace

---

## 🔇 Step 3: Transition to no-music (critical skill)

You should **actively turn music OFF** when:

### 1. You enter language-heavy work

* reading dense material
* writing
* understanding new concepts

👉 Signal:

> “I need to reread this sentence”

---

### 2. You’re deeply locked in

Sometimes:

* focus becomes stable
* music becomes unnecessary load

👉 Signal:

> “I’m fully inside the task”

Turn it off or switch to white noise

---

### 3. Cognitive strain increases

If you feel:

* mental fatigue
* confusion
* slower thinking

👉 Music is now interference, not support

---

# 🧭 3. A Clean Daily Flow

Here’s how this looks in practice:

---

### 🟢 Example: Coding session

1. Start → **Electronic Focus** (5–10 min)
2. Switch → **Deep Focus**
3. Hit difficult logic → **lower volume or silence**
4. Continue

---

### 🔵 Example: Lecture

1. Start → **Electronic Focus**
2. Switch → **Lo-Fi Beats**
3. If drifting → slightly increase volume or stimulation
4. If struggling to understand → **pause music**

---

### 🔴 Example: Restless day

1. Start → **Electronic Focus**
2. Switch → **Brain Food**
3. Work in short bursts
4. If overwhelmed → pause + silence + reset

---

### 🟣 Example: Reading-heavy

1. Start → **Electronic Focus**
2. Begin reading
3. Within minutes → **turn OFF music or switch to white noise**

👉 Reading rarely pairs well with music long-term

---

# ⚖️ 4. Subtle but Important Rules

## Rule 1: Don’t mix playlists randomly

Each one should become a **mental trigger**

---

## Rule 2: Volume matters

* Too loud → distraction
* Too quiet → ineffective

👉 Aim for *barely noticeable but present*

---

## Rule 3: Don’t chase perfect songs

Familiarity > novelty

---

## Rule 4: Music is a tool, not a crutch

The goal is:
👉 easier entry into focus
not
👉 permanent dependency
